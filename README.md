# TDA_cages

This repository contains the scripts used in "Identifying porous cages subsets in the Cambridge Structural Database using topological data analysis" by Aurelia Li, Rocio Bueno-Perez and David Fairen-Jimenez, Adsorption and Advanced Materials Group (aam.ceb.cam.ac.uk), led by David Fairen-Jimenez from the Department of Chemical Engineering and Biotechnology, University of Cambridge.
